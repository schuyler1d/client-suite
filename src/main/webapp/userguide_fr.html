Guide de l'utilisateur d'Annotation d'images

Bienvenue sur le service Annotation d'image. Ce service peut être utilisé pour créer, modifier et supprimer des annotations pour les images disponibles. Chaque annotation peut être rattachée à une partie déterminée de l'image, i.e. vous pouvez définir une aire de forme rectangle, elliptique ou polygonale sur l'image et y rattacher votre annotation.

Créer un commentaire

Si vous êtes inscrits et connectés, vous pouvez créer vos propres commentaires, les modifier et les supprimer. Vous pouvez aussi répondre à un commentaire déjà présent. Dans la figure ci-dessous, vous voyez deux possibilités : "Commenter l'image" et "Commenter une partie de l'image". Vous pouvez aussi choisir de rendre vos commentaires publics ou de les garder privés : les annotations publiques sont visibles par tout le monde ; les commentaires privés ne sont visibles que par vous.

Si vous choisissez "Commenter l'image", vous verrez apparaître dans la même fenêtre un formulaire pour entrer le titre et le texte de votre commentaire.

Si vous choisissez "Commenter une partie de l'image", vous verrez non seulement apparaître le formulaire d'annotation, mais aussi une palette d'outils pour sélectionner une partie de l'image. Dans l'exemple ci-dessous, c'est un cercle brun. Vous pouvez déplacer la palette par glisser-déposer, et vous pouvez modifier la taille de la sélection en glissant-déposant les points de sélection. La forme, l'épaisseur du trait et la couleur peuvent être modifiées en utilisant les boutons-radio correspondant et la palette de couleurs. Si vous choisissez de créer une aire polygonale, vous devrez cliquer sur l'image à chaque angle du polygone. Les points seront reliés entre eux dès que vous aurez cliqué. Placez le dernier point juste à côté du premier pour fermer le polygone. Si vous voulez tracer une forme libre, vous devrez cliquer pour démarrer le tracé, puis déplacer la souris. Cliquer une seconde fois pour terminer le tracé.

Vous pouvez utiliser les symboles et sur l'image ou la molette de la souris pour zoomer. Vous pouvez faire glisser l'image pour voir d'autres parties. Quand vous avez fini, il suffit de cliquer sur "Sauvegarder" pour sauvegarder votre commentaire, ou sur "Annuler" pour supprimer vos modifications.

Répondre à un commentaire

En plus de pouvoir ajouter de nouveaux commentaires, vous pouvez aussi répondre à des commentaires déjà présents. Pour ce faire, cliquez sur "Répondre" ou "Répondre avec fragment" pour ajouter un commentaire textuel ou graphique. Le reste du processus pour ajouter un commentaire est le même que ci-dessus.

Modifier des commentaires

Pour modifier un commentaire, le processus est presque identique. Vous devez cliquer sur "Modifier" sous l'annotation dans la liste des annotations. De nouveau, selon que vous modifiez un commentaire portant sur l'ensemble de l'image ou seulement sur une partie de l'image, vous voyez apparaître le formulaire d'annotation avec le titre et la description ou le formulaire avec la palette d'outils. Quand vous avez fini, cliquez sur "Sauvegarder" pour sauvegarder vos modifications, ou sur "Annuler" pour les supprimer.

Nota bene : vous ne pouvez modifier que les commentaires que vous avez créés vous-même et seulement si aucune réponse n'y a été faite.

Supprimer des commentaires

Tout commentaire peut être supprimé en cliquant sur le bouton correspondant "Supprimer". Cette suppression ne peut être effectuée que par l'auteur original du commentaire.

Nota bene : vous ne pouvez supprimer que les commentaires que vous avez créés vous-même et seulement si aucune réponse n'y a été faite.
